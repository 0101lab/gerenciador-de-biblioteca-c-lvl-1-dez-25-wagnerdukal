# Gerenciador de Biblioteca — C++ (Nível Iniciante)

Implemente um programa de terminal em C++ para gerenciar
uma pequena biblioteca.

## Funcionalidades
- Cadastrar livros (título e autor)
- Listar livros
- Buscar livro pelo título
- Menu interativo

## Observações
- Uso de structs ou classes é esperado
- Não é necessário persistência em arquivo

## Compilação
```bash
g++ main.cpp -o biblioteca
./biblioteca
